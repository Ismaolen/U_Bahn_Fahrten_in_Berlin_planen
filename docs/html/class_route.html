Detailed Description

class specializing class edge into a unidirectional route, with distance as a member variable distance is also used to define the weight of the route, thereby influencing the Dijkstra search for the shortest path

Dieser Klasse ist einer erweiterun der Klasse Edge die zu den strecken die Line und distanz und dauer hinzufügt.
